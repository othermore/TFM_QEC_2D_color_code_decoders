\begin{notebox}{Source code and repository structure}
Chapters \ref{chap:modeling_quantum_error} to \ref{chap:simulating_2d_color_codes} show the result of multiple simulations and code execution. 

Small code snippets are shown to illustrate how the different tools and libraries work. The complete code required to execute the simulations presented in this work is available in a git repository at \url{https://github.com/othermore/TFM_QEC_2D_color_code_decoders}. 

The code is located in the \texttt{src} directory. Each chapter has its own directory and python notebook. The specific code for each simulation is located in separate files (starting with \texttt{s} and the number of the simulation) and included in the notebook. 

Libraries and resources required for multiple simulations are located in the directory \texttt{src/lib}.
\end{notebox}
